macOS安装Python教程

更新时间：August 22, 2018

作者：Windson Yang 著作权归作者所有。商业转载请联系作者获得授权，非商业转载请注明出处。

以下步骤极其简单，不要担心😊

安装Python

  1. 访问Python下载地址，如下图：

  2. 点击上图蓝色框中的链接，就像下载普通文件一样，这个时候浏览器会自动开始下载安装包，或者电脑中的默认下载软件会弹出来，请你选择下载目的地，再点击确认。下载完毕之后双击该安装包（一般情况下在下载目录中），出现下图，点击继续：

  3. 下图包括软件的基本介绍，点击继续。

  4. 下图包括软件的使用协议，点击继续，接着软件会弹出一个小窗口，需要你同意协议，点击同意按钮即可。

  5. 下图选择安装的地方，macOS一般只有一个盘，点击继续即可。

  1. 出现下图后，点击安装，软件会弹出一个小窗口，要求输入管理密码，你输入管理密码之后，回车，然后软件会开始安装，稍等几分钟。

  1. 显示下图界面则代表安装成功了。如果没有安装成功的话请仔细核对以上的步骤重新安装一次。
